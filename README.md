# Open Veterinarian 🐾

_The ultimate open-source assistance app for Veterinarians._

## 🎹🦞 Vision
Open Veterinarian is designed to be the definitive digital toolkit for veterinary professionals. Built with the same precision as a synthwave bassline and the reliability of a cybernetic lobster, this app bridges the gap between complex medical data and rapid, on-the-go clinical decision making.

## Core Features (Finalized Roadmap v1.7)
- **Responsive Clinical HUD:** Optimized for Mobile and Tablet (Split-screen multi-tool layout).
- **Emergency Triage HUD:** Interactive A-B-C-D-E checklists for rapid assessment.
- **Quick Reference Engine:** Instant access to drug dosages, normal vitals, and pathologies.
- **Surgical Note Taker:** Voice-to-text dictation and text-to-speech playback for surgical records.
- **Blood Gas Interpreter:** Rapid acid-base analysis and compensation logic.
- **Imaging & Anatomy:** Detailed X-Ray/Ultrasound guides and visual anatomy atlas.
- **Dose & Fluid Suite:** High-precision calculators for drugs, fluids, additives, and CRI.
- **Emergency HUD:** Rapid crash cart math with an integrated 110 BPM CPR Metronome.
- **Client Communication:** Professional PDF reports and educational handouts for pet owners.
- **Cloud Sync & Backup:** Multi-device synchronization (Supabase) and local Hive backups.

## Tech Stack
- **Frontend:** Flutter (Dart) - Responsive HUD.
- **State Management:** Riverpod.
- **Persistence:** Hive (NoSQL local storage).
- **Cloud Engine:** Supabase.
- **Audio/Voice:** flutter_tts & speech_to_text.
